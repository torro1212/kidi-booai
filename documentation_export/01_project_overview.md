# 01_project_overview.md

## Project Name
**KidCraft AI** (Version 2.1)

## Tool Description
KidCraft AI is a web-based application that empowers users to create personalized, illustrated children's books using advanced AI. It generates coherent stories with consistent characters, custom art styles, and optional narration (Text-to-Speech), allowing for a seamless "idea-to-book" experience.

## Target Users
*   **Primary:** Parents creating stories for their children.
*   **Secondary:** Teachers and Educators creating custom learning materials.
*   **End Users:** Children aged 3 to 16.

## Book Types Supported
*   **Standard Illustrated Books:** Full-page illustrations with text.
*   **Comic Books:** specialized 4-panel comic strip layouts.
*   **Sequels:** Continuity-aware stories that maintain character consistency from previous books (via PDF/JSON analysis).
*   **Audio Books:** Narrated stories with character-specific voices.

## Platforms
*   **Web Application:** Fully responsive Next.js application.
*   **Mobile Optimized:** Specific features for mobile usage (preventing sleep mode during generation, touch-friendly UI).

## Main Value Proposition
"Making learning magical" by turning simple inputs (a topic, an age group) into professional-quality, educational, and entertaining books with high visual consistency and instant export capabilities.
